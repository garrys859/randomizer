Política de Privacidad y Uso de la API de YouTube

En Robinexus, nos comprometemos a operar de manera transparente y responsable, respetando la privacidad de nuestros usuarios y cumpliendo con los términos de servicio de las plataformas que utilizamos. Este sitio web utiliza la API de YouTube para mostrar videos incrustados y "randomizar" las playlist que tengan los usuarios dentro de la plataforma Youtube. Al utilizar nuestro sitio web, aceptas las siguientes condiciones relacionadas con el uso de la API de YouTube:

Cumplimiento de los Términos de Servicio

El uso que hacemos de la API de YouTube se rige por:

  • Los Términos de Servicio de YouTube: Puedes consultarlos en https://www.youtube.com/t/terms.
  • La Política de Privacidad de Google: Puedes consultarla en https://policies.google.com/privacy.

Te instamos a leer detenidamente estos documentos para comprender tus derechos y responsabilidades al utilizar YouTube y los servicios de Google.

Datos Recopilados y su Uso

En Robinexus no recogemos datos personales pero si datos no personales como el ID de playlist de los usuarios.

Para ofrecer la funcionalidad de crear listas de reproducción aleatorias, recopilamos únicamente los IDs de las listas de reproducción de YouTube que los usuarios ingresan. Esta información se utiliza exclusivamente para generar la aleatorización y no se comparte con terceros ni se utiliza para ningún otro fin.

Cookies y Tecnologías de Seguimiento

Este sitio web puede utilizar cookies para mejorar la experiencia del usuario y recordar preferencias (como canciones marcadas como favoritos y playlist guardadas). Puedes configurar tu navegador para rechazar las cookies, pero esto podría afectar la funcionalidad del sitio. Puedes consultar nuestra cookies en cualquier momento, desde Robinexus recomendamos aceptarlas.

Enlaces a Otros Sitios Web

Nuestro sitio web puede contener enlaces a otros sitios web, incluyendo YouTube. No nos hacemos responsables de las políticas de privacidad de estos sitios. Te recomendamos revisar las políticas de privacidad de cada sitio que visites.

Cambios a esta Política

Podemos actualizar esta política de privacidad periódicamente. Te notificaremos sobre cualquier cambio importante publicando una nueva versión en esta página.
